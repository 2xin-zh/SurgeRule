🌚 2 = custom,1.2.3.4,443,aes-256-cfb,password,https://github.com/jinyu121/SurgeRule/raw/master/SSEncrypt.module, ota=true
[Proxy Group]
🚀 Proxy = select, 💊 Direct, 🌞 Auto, 🌚 Auto, ⚖ Select
🌐 Proxy = select, 💊 Direct, 🚀 Proxy
🇨🇳 Proxy = select, 💊 Direct, 🚀 Proxy

🌞 Auto = url-test, 🌞 1, 🌞 2, url = http://www.gstatic.com/generate_204

⚖ Select = select, 🌞 1, 🌞 2, 🌚 1, 🌚 2

配置说明

在[Proxy]中，建议保留如下 Proxy ：

💊 Direct = direct
直连规则

在[Proxy Group]中，必须保留如下 Group ：

🚀 Proxy
一般代理
🌐 Proxy
兜底全局代理
🇨🇳 Proxy
国内网站访问规则
🍎 Proxy
苹果服务专用规则

更详的细配置说明，请参阅官方英文版《 Surge 使用手册 》
